    set(CMAKE_VERBOSE_MAKEFILE ON)

cache 变量

normal variable: 普通变量，局部作用域
cache variable: 缓存变量，全局变量, 同一个CMAKE功能任何地方都可以用

set( … CACHE [FORCE])
variable：变量名称
value：变量值列表
CACHE：cache变量的标志
type：变量类型，取决于变量的值。类型分为：BOOL、FILEPATH、PATH、STRING、INTERNAL
docstring：必须是字符串，作为变量概要说明
FORCE：强制选项，强制修改变量值
